### Installation

Fluig速 AngularJS requires [Node.js](https://nodejs.org/) v4+ to run.

Install the dependencies and devDependencies and start the server.

```sh
$ cd angular-fluig
$ npm install -d
$ npm start
```

Open /examples/index.html to view the demo
